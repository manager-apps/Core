## Installing
Overwrite flag is by default false, so the newer version will be installed
without overwriting configuration.

DciAgentService-1.0.1-Setup.exe /VERYSILENT /NORESTART

##
Overwrite existing installation including configuration

DciAgentService-1.0.1-Setup.exe
  /VERYSILENT
  /NORESTART
  /OVERWRITECONFIG=1
  /SERVERURL="http://..."
  /AREANAME="A"
  /AGENTNAME="X"
  /CLIENTSECRET="S"
